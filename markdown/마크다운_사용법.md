## Paragraphs and Text Decoration

이탤릭체 / 볼드체 / 취소선 사용하기
```
*텍스트*: 이탤릭체
cat is *cute*.  

**텍스트**: 볼드체
cat is real **cute**.  

***텍스트*** - 이탤릭&볼드체
cat is real ***cute***.  
```
cat is *cute*.
cat is real **cute**.
cat is real ***cute***.
<br>

```
 _텍스트_: 이탤릭체
cat is _cute_.

__텍스트__: 볼드체
cat is real __cute__. 

___텍스트___: 이탤릭&볼드체
cat is real ___cute___. 
```
cat is _cute_.
cat is real __cute__. 
cat is real ___cute___. 

<br>
```
~~텍스트~~: 취소선
cat is ~~cute~~. 
```
cat is ~~cute~~.

## Headings in Markdown

```

```
