# Functional Programming

## Function

함수는 수학이라는 학문과 함께 오랜 시간 우리가 다루어 온 존재입니다.
이미 충분한 검증이 이루어진 이론이기 때문에 신뢰할 수 있습니다.
더욱 견고한 구조를 구축할 수 있다.



**javascript에서는 input -> output**

수학에서 함수랑 맵핑이랑 동일하다. 인풋과 아웃풋으로 이루어져서 맵핑이라 한다.

## Pure Function

수학적인 정의의 함수를 따른다.
자신감이 많아진다. 인풋 -> 아웃풋이 일정하기 때문에 디버깅 할 필요가 없다.
잘 들어가기만 하면 된다.
